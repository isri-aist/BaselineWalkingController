18 ### Installation procedure
19 It is assumed that ROS is installed.
20 
21 1. Setup catkin workspace.
22 ```bash
23 $ mkdir -p ~/ros/ws_ccc/src
24 $ cd ~/ros/ws_ccc
25 $ wstool init src
26 $ wstool set -t src isri-aist/QpSolverCollection git@github.com:isri-aist/QpSolverCollection.git --git -y
27 $ wstool set -t src isri-aist/NMPC git@github.com:isri-aist/NMPC.git --git -y
28 $ wstool set -t src isri-aist/CentroidalControlCollection git@github.com:isri-aist/CentroidalControlCollection.git --git -y
29 $ wstool update -t src
30 ```
31 
32 2. Install dependent packages.
33 ```bash
34 $ source /opt/ros/${ROS_DISTRO}/setup.bash
35 $ rosdep install -y -r --from-paths src --ignore-src
36 ```
37 
38 3. Build a package.
39 ```bash
40 $ catkin build centroidal_control_collection -DCMAKE_BUILD_TYPE=RelWithDebInfo --catkin-make-args all tests
41 ```

46 ### Methods based on bipedal dynamics
47 The CoM and ZMP trajectories are planned according to the ZMP reference trajectory and the ZMP region boundaries as inputs, which are determined from a given footstep sequence (i.e., the position and timing of the foot landings). The CoM velocity is jumped by emulating a disturbance during motion.
