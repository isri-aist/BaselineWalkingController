18 ### Installation procedure
19 It is assumed that [ROS](http://wiki.ros.org/ROS/Installation) and [mc_rtc](https://jrl-umi3218.github.io/mc_rtc/tutorials/introduction/installation-guide.html) is installed.

21 1. Setup catkin workspace.
22 ```bash
23 $ mkdir -p ~/ros/ws_bwc/src
24 $ cd ~/ros/ws_bwc
25 $ wstool init src
26 $ wstool set -t src isri-aist/QpSolverCollection git@github.com:isri-aist/QpSolverCollection.git --git -y
27 $ wstool set -t src isri-aist/NMPC git@github.com:isri-aist/NMPC.git --git -y
28 $ wstool set -t src isri-aist/CentroidalControlCollection git@github.com:isri-aist/CentroidalControlCollection.git --git -y
29 $ wstool set -t src isri-aist/BaselineWalkingController git@github.com:isri-aist/BaselineWalkingController.git --git -y
30 $ wstool update -t src
31 ```
32 
33 2. Install dependent packages.
